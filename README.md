# Cookie Notifier

QoL mod that switches the browser tab favicon in Cookie Clicker to indicate golden and wrath cookie spawns.

## Usage

1. Install [Cookie Clicker Script Extender (CCSE)](https://klattmose.github.io/CookieClicker/CCSE-POCs/) if not already in use.
2. Load the mod via Cookie Clicker Mod Manager or CCSE using this script URL: https://cdn.jsdelivr.net/gh/MogensenJesse/cookie-notifier@main/cookieNotifier.js
3. Make sure your adblocker is set to whitelist the jsDelivr URL if you encounter loading issues (check for console warnings).
